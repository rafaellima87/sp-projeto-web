# Silva’s Parrilla

Projeto front-end desenvolvido para a disciplina do curso técnico do SENAC TECH.

O sistema foi construído utilizando HTML, CSS e JavaScript, com apoio do HTMX, sem a utilização de backend.

Objetivo do Projeto

A proposta do projeto é desenvolver um sistema de treinamento interno para os colaboradores do restaurante Silva’s Parrilla.

O site apresenta conteúdos em formato audiovisual e textual. Ao final de cada conteúdo, o colaborador realiza um teste de conhecimento. Ao acertar todas as questões, ele tem acesso a uma tela para geração de certificado (no momento ainda não gera o certificado).

Em uma futura implementação com backend, será possível incluir no certificado:

- Nome do usuário

- Trilha concluída

- Data de conclusão

# Área Administrativa

O sistema possui uma área exclusiva para o administrador (gestor), com acesso a funcionalidades específicas.

Como melhoria futura, a ideia é permitir que o próprio administrador:

- Cadastre novos conteúdos

- Crie e gerencie questionários

- Trabalhe com conteúdos e quizzes dinâmicos, conforme a necessidade da empresa (Silva's Parrilla)

Como Executar o Projeto:

Abra o arquivo index.html em qualquer navegador web moderno.

# Observações

O login e os quizzes são simulados utilizando JavaScript.

Não utiliza banco de dados nem armazenamento local (localStorage).
